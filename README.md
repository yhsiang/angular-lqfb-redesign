angular-lqfb-redesign
=====================

angular frontend for Liquid Feedback (core+pgrest)

Installation
```
$ npm i
```
Start Server
```
$ npm start
```

By now the REST call is just a call to a local json file.

Translate
======================

using lazy.js

Pirate Feedback: frontend / locale / *.lua
to
app / locale / *.json
